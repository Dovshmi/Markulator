# Markulator Web

גרסת React/Vite עברית וימין־לשמאל של Markulator.

## סדר במצב סבולת ±

הסדר בטופס ובתוצאה הוא:

1. סבולת חיובית / Tol+
2. מידה נומינלית / Nominal
3. סבולת שלילית / Tol-

## הרצה מקומית

```bash
cd web
npm install
npm run dev
```

## בנייה לפרודקשן

```bash
npm run build
```

## פריסה ב־Vercel

- Root Directory: `web`
- Build Command: `npm run build`
- Output Directory: `dist`
