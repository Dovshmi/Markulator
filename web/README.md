# Markulator Web

גרסת React/Vite עברית וימין־לשמאל של Markulator.

## הרצה מקומית

```bash
cd web
npm install
npm run dev
```

## בנייה לפרודקשן

```bash
npm run build
```

## פריסה ב־Vercel

- Root Directory: `web`
- Build Command: `npm run build`
- Output Directory: `dist`

גרסת הדסקטופ של Markulator נשארת בתיקיית השורש של הפרויקט, והגרסה הזו חיה בנפרד בתיקיית `web`.
